Sobre este Proyecto

El proyecto consiste en una simulación de una página web de un cine, diseñada para ofrecer a los usuarios una experiencia similar a la de un sitio real. La página incluye varias secciones clave: cartelera de películas, detalles de cada película, historia del cine e información del desarrollador.

Tecnologías Utilizadas

  • HTML: Se utiliza para estructurar el contenido de la página, incluyendo la creación de elementos como tarjetas para las películas y contenedores para las distintas secciones de la página.
  • CSS: Se usa para estilizar la página y mejorar la experiencia del usuario a través de diseños responsivos, efectos visuales y animaciones.
  • JavaScript: Se emplea para agregar interactividad a la página, como la actualización dinámica del contenido, la integración de páginas web externas, entre otros.
